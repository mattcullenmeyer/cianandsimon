## Architecture

This is a React 19 app using **Panda CSS** for styling and **Park UI** component patterns built on **Ark UI** headless components.

### Styling Stack

- **Panda CSS**: CSS-in-JS with static extraction. Config in `panda.config.ts`, generates `styled-system/` directory
- **Park UI**: Component recipe patterns (not installed as a package, patterns are in `src/theme/`)
- **Ark UI** (`@ark-ui/react`): Headless component primitives with accessibility built-in

### Key Directories

- `src/components/ui/` - UI components that wrap Ark UI with Panda CSS styling
- `src/theme/` - Panda CSS theme configuration
  - `recipes/` - Component style recipes (variants, slots)
  - `colors/` - Semantic color scales (teal, slate, red, green)
  - `tokens/` - Design tokens (colors, shadows, durations, z-index)
- `styled-system/` - Generated by Panda CSS (do not edit manually)

### Component Pattern

Components follow the Park UI pattern: wrap Ark UI components with `createStyleContext` for styling:

```tsx
import { Timer } from '@ark-ui/react/timer';
import { createStyleContext } from 'styled-system/jsx';
import { timer } from 'styled-system/recipes';

const { withProvider, withContext } = createStyleContext(timer);

export const Root = withProvider(Timer.Root, 'root');
export const Item = withContext(Timer.Item, 'item');
```

### Semantic Tokens

Use semantic tokens for colors instead of raw values:
- `fg.default`, `fg.muted`, `fg.subtle` - Foreground/text colors
- `colorPalette.solid.bg`, `colorPalette.subtle.fg` - Component color palette tokens
- `border` - Border color

### Typography

Use `textStyle` prop for consistent font-size + line-height pairings:
```tsx
<Text textStyle="lg">...</Text>  // Available: xs, sm, md, lg, xl, 2xl, 3xl, 4xl, 5xl, 6xl, 7xl
```

### Path Alias

`@/*` resolves to `./src/*`
